# BalanceFormatter

 This plugin works with any plugins using PlaceholderAPI
 
 ![Preview](https://i.imgur.com/eYCIBF5.png)
 
 **Placeholders:**
  - %balanceformatter_formated% 
  - {placeholderapi_balanceformatter_formated} *(For plugins like FeatherBoard)*
  

 **Dependencies:**
  - Vault
  - PlaceholderAPI
  - A Economy Plugin
