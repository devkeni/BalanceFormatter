# BalanceFormatter

 This plugin works with any plugin using PlaceholderAPI
 
 ![Preview](https://i.imgur.com/7aN1IiG.png)
 
 **Placeholders:**
  - %balanceformatter% 
  - {placeholderapi_balanceformatter} *(For plugins like FeatherBoard)*
  

 **Dependencies:**
  - Vault
  - PlaceholderAPI
  - A Economy Plugin
